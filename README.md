
# 🛍️ Myntra Sales Analysis Dashboard | Power BI

An interactive **Power BI Sales Dashboard** developed to analyze Myntra's sales performance, customer behavior, product performance, and discount trends. This project transforms raw transactional data into meaningful business insights through data modeling, DAX measures, and interactive visualizations.

## 🚀 Project Highlights

- 📊 Interactive multi-page dashboard
- 📈 Sales trend analysis
- 👕 Product performance analysis
- 👥 Customer insights
- 🏷️ Discount analysis
- 🌍 State-wise sales visualization
- 📌 Dynamic KPI cards and insights
- 🎯 User-friendly navigation with slicers

## 📌 Key KPIs

- 💰 Total Revenue
- 🛒 Total Orders
- 👥 Total Customers
- ⭐ Average Rating
- 🏷️ Average Discount %
- 🏆 Top Brand
- 👕 Top Category

## 📊 Dashboard Pages

- Overview
- Product Performance
- Customer Analysis
- Discount Analysis

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations

## 📈 Business Insights

- Identified top-performing brands and product categories.
- Analyzed monthly sales trends and regional performance.
- Evaluated the impact of discounts on revenue.
- Compared customer purchasing patterns across different states.
- Created dynamic KPIs and interactive dashboards for better decision-making.
